# Page tampon SOTELEM INDUSTRIE (version mobile-friendly + LinkedIn)

Contenu du dossier :

- `index.html` : page tampon principale, responsive (smartphone), fond rouge, horaires, bouton LinkedIn.
- `assets/bandeau-sotelem-activites.png` : bandeau avec les pictogrammes Découpe / Soudure / Pliage / Peinture.

## Utilisation

1. Déposez tout le dossier sur un hébergement web (ou en local).
2. Ouvrez `index.html` dans un navigateur pour tester (ordinateur + smartphone).
3. Vous pouvez pointer vos QR codes vers l’URL de cette page.
